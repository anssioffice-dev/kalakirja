Google Sheets -yhteys

Kirjaa saaliit automaattisesti omaan Google-taulukkoon. Taulukko näkyy vain sinulle, kun käytät Apps Script -kirjausta.

  1. Luo uusi Google-taulukko ja lisää otsikkorivi: Aika, Päivä, Laji, Pyyntitapa, Paino(g), Pituus(cm), Sijainti, Sää, Koordinaatit
  2. Avaa Laajennukset → Apps Script
  3. Liitä alla oleva koodi ja tallenna
  4. Klikkaa Ota käyttöön → Uusi käyttöönotto → Web-sovellus
    Suorita nimellä: Minä · Käyttöoikeus: Vain minä
  5. Kopioi käyttöönoton URL alle

Apps Script -koodi:

function doPost(e) { const s = SpreadsheetApp .getActiveSpreadsheet() .getActiveSheet(); const d = JSON.parse(e.postData.contents); s.appendRow([ d.time, d.date, d.species, d.method||'', d.weight, d.length||'', d.location, d.weather, d.coords||'' ]); return ContentService .createTextOutput('OK'); }
